News Report on Volunteer Activity

A volunteer activity was organized by the Student Union in our University on 5th June last week. The activity was designed to visit and help the elderly people in the neighborhood.

In addition, they played games with the elderly people and delivered a series of art programs to the welcoming seniors.

The society and family must give priority to helping the elderly, which is indispensable for not only their mental and physical health but also for the long-term development of the society.

浏览器扩展的使用

2021-02-26

考虑到谷歌应用商店对国内用户的不用好，所有本篇以添加Edge扩展介绍，Edge是win10自带，确保更新为最新版本，以下是下载链接，选择windows进行安装。

安装插件首先确保开发人员模式的打开，并且允许来自其他应用商店的扩展，这样便可以安装谷歌应用商店的扩展。

首先推荐一个教(Tampermoney)油猴的扩展，它是一个插件的集合。

它集合了许多插件于一体，使用起来特别方便。如视频vip 音乐vip 翻译器 csdn等各大网站插件，非常强悍！！

以下Tampermoney使用的教程：

  • 选择Tampermoney（油猴）图标，获取新脚本

  • 然后便有许多链接的选择，这里选择GreasyFork 。因为里面包含中文简体，对用户友好。

  • 直接搜索想要的内容，这里搜索vip用于破解看视频，便可获得很多脚本

  • 选择第一个懒人专用进行安装

  • 安装以后便可返回，再次点击油猴图标，选择管理脚本查看是否安装成功

现在进行懒人专用脚本的使用教程：

  • 任意打开一个网页的视频播放器，这里选择腾讯视频

  • 选择vip会员，可以看到这里并没有登录会员账号

  • 任意点击一个vip视频进行播放，可以看到视频左侧出现了一个vip图标，接下来就可以任意选择一个站点进行解析播放了！！！

油猴还组合了许多实用的插件，就不一一列举了，可以自行添加下载和vip视频添加的方式一样！

接下来推荐一个标签页：Infinity 新标签页，同上油猴的安装。它是一个标签页的布局，可以让页面变得非常的漂亮。

插件的使用因人而异，适合自己的就是最好的。插件也并不是越多越好，插件安装过大，会导致浏览器页面的卡顿，适当安装。

这里展现我安装的插件。

edge浏览器同步了谷歌的插件非常的实用，不用翻外网，用的非常舒服。并且浏览器自带了许多功能，如朗读功能，右键页面—>大声朗读，可以让学习更加方便！

学习插件的使用，让浏览器更加的强大！！

Hexo + github 基础搭建 和 使用教程

2021-01-19

第一篇博客

首先使用hexo搭建个人博客必须准备好工具

  • 安装node.js （建议版本12.14 版本太高之后可能产生错误）

  • 安装Git

  • 注册GitHub账号

  • 安装Hexo

  • 连接Github与本地

  • 写文章和发布文章

    对应的安装可以去各自的官网进行安装下载

安装Hexo

在电脑对应位置新建一个文件夹，用来存放自己的博客文件，如D盘，进行命名并新建，这里以blog命名
D:\blog
新建完成后，在该文件夹下，右键点击Git Bash Here（前提是已经安装好了Git）
输入npm i hexo-cli -g安装Hexo ，报错可以忽视
安装完成后 可输入hexo -v 查看是否安装成功
接下来输入hexo init初始化文件夹 接下来输入npm install安装必备组件
输入hexo g生成静态网页，然后输入hexo s打开本地服务器，接着输入localhost:4000便可访问第一个博客了

总结

  • 输入 npm i hexo-cli -g 安装Hexo
  • ​ hexo -v 查看是否安装成功
  • ​ hexo init 初始化文件夹
  • ​ npm install 安装必备组件
  • ​ hexo g 生成静态网页
  • ​ hexo s 打开本地服务器

配置到github

在注册github之后 登录账号，首先找到右上角的“➕”号 点击，选择New repository进行新仓库的创建

​

首先填写Repository name信息 一定按照当前的用户名填写 username.github.io （username为你的账号）
Description选填 进行描述 后勾上Add a README file文件，点击创建仓库Create repository 由于本人已经
创建过了仓库 所以标红

配置SSH key

由于提交代码需得到github权限，但直接使用账号密码不安全，所以使用ssh key来解决本地和服务器的连接问题
$ cd ~/. ssh #检查本机已存在的ssh密钥
如果提示：No such file or directory 说明你是第一次使用git。
ssh-keygen -t rsa -C “邮件地址” 邮件地址即为你注册时使用的地址
然后连续3次回车，最终会生成一个文件在用户目录下，打开用户目录，
找到.ssh\id_rsa.pub文件，记事本打开并复制里面的内容，
打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key：

将刚复制的内容粘贴到key那里，title随便填，保存。

测试是否成功

$ ssh -T git@github.com
如果提示Are you sure you want to continue connecting (yes/no)?，输入yes，然后会看到

继续配置个人信息
$ git config –global user.name “liuxianan”// 你的github用户名，非昵称
$ git config –global user.email “xxx@qq.com“// 填写你的github注册邮箱

接下来打开博客根目录下的_config.yml文件，这是博客的配置文件，在这里你可以修改与博客相关的各种信息。
修改最后一行的配置：
deploy:
type: git
repository: https://github.com/qq175/qq175.github.io
branch: master
repository修改为你自己的github项目地址。

总结：

  • 先github创建仓库需注意仓库名
  • 配置SSH key
  • 配置个人信息
  • 修改博客根目录下的_config.yml文件

部署与查看

首先在博客根目录下安装一个扩展npm install hexo-deployer-git –save
接着输入 hexo d 即可导入到github 此时刷新github页面即可发现多了许多文件夹
网址栏输入 username.github.io 进行访问你的第一个博客吧！！
如果hexo d命令已经导入成功了，但是github没显示的话，注意是否有将master改为默认的Branches

总结：

  • 输入命令npm install hexo-deployer-git –save安装扩展
  • hexo d命令进行导入
  • 将github中的Branches 默认改为配置文件中的 master
  • 输入网址username.github.io进行查看 (username自身账户)
